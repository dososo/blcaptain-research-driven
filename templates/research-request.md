# Research Request

Copy this prompt and replace the bracketed fields before asking an Agent to investigate.

```text
Use blcaptain-research-driven to investigate: [question]

Decision this research should support: [decision]
Required scope: [date, version, jurisdiction, environment, or population]
Use primary sources, cross-verify key conclusions, actively seek counterevidence, and grade each conclusion as Confirmed, Probable, Speculative, or Unknown.
Stop only after the explicit STOP conditions hold, and name any remaining gaps.
```
